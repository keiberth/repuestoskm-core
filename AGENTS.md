# Proyecto RKM

## Descripción
Plugin WordPress/WooCommerce para gestión de catálogo y pedidos tipo ERP.

## Estado actual
- Panel de usuario funcional
- Catálogo en desarrollo
- Pedidos en progreso

## Objetivo actual
Agregar modal de producto en el catálogo con:
- galería de imágenes
- información detallada
- UI moderna

## Estructura
- templates/: vistas del frontend
- assets/js/: interacción
- assets/css/: estilos
- includes/: lógica PHP

## Reglas
- Usar prefijo rkm-
- No romper funcionalidades existentes
- Evitar estilos por defecto de WooCommerce
- Mantener diseño limpio y profesional

## UX objetivo
Tipo MercadoLibre:
- cards modernas
- interacción rápida
- modal elegante
